## 安装electon

1. 　　
　　pnpm：

　　　　pnpm config set electron_mirror "https://npm.taobao.org/mirrors/electron/"

　　npm：

　　　　 npm config set electron_mirror "https://npm.taobao.org/mirrors/electron/"

2.

　　pnpm install electron

　　npm install electron
